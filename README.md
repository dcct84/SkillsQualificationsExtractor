# Keywords Extractor related to skills and qualifications

This repository contains the code used to extract keywords from text in the attempt to identify skills and qualifications, most commonly required. This code is based on Kavita Ganesan's Extracting Keywords with TF-IDF and Pythonâ€™s Scikit-Learn tutorial see url: https://kavita-ganesan.com/extracting-keywords-from-text-tfidf/#.XVqzMuhKibi

The toy dataset attached is a list of Terms of Reference published in the IDB site when recruiting personnel. The purpose of this project is to see if a list of skills can be extracted as keywords.
